# **App Name**: Simple Hello

## Core Features:

- Display Hello: Displays the word 'Hello' on the screen.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a calm, focused feel.
- Background color: Light Indigo (#E8EAF6) to ensure good contrast.
- Accent color: Vibrant Yellow (#FFEB3B) to highlight the displayed word.
- Font: 'Inter', a grotesque-style sans-serif, will be used for the displayed word, ensuring a clean and modern look.
- Center the text 'Hello' in the middle of the screen for immediate visibility.
- Use a subtle fade-in effect when the page loads.
